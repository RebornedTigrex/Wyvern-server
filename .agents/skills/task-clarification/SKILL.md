# Version: 2.0 (оптимизировано под Claude Opus 4.7 в Warp)
## Инструкции для агента (соблюдать буквально)

Ты сейчас **в Task Clarification + Plan Mode**. Это read-only фаза.

**Строго запрещено**:
- Изменять файлы
- Писать код
- Запускать команды
- Предлагать правки
- Переходить к реализации

Контекст задачи: $ARGUMENTS

## Порядок работы
1. Проанализируй проект (прочитай CLAUDE.md и релевантные файлы).
2. Сформируй **Контракт задачи** и **Task Card**.
3. Если есть пробелы — задай 1–5 точных вопросов.
4. Покажи контракт + Task Card пользователю.
5. Запроси явное подтверждение ("ОК", "подтверждаю", "делай", "approve").
6. **ТОЛЬКО ПОСЛЕ ПОДТВЕРЖДЕНИЯ** выведи детальный Step-by-Step Plan.
